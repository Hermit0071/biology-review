# biology-review
# 微观有光｜浙科版高中生物学考复习

## 启动

双击 **生物学考复习.html** 即可直接打开程序，不需要启动命令、终端窗口或本地服务器。

程序完全离线运行，不需要登录或联网。学习记录保存在当前浏览器中。

## 主要功能

- **按课复习**：两册教材共 9 章、33 节课，每课包含目标、概念、过程、教材实验、易错辨析、学考提示和教材页码。
- **专题复习**：8 个跨章节专题，以及遗传棋盘格、自由组合、实验变量和进化时间轴等互动工具。
- **模型馆**：13 组可旋转、缩放、拆解、播放和标注的三维教学模型。
- **我的进度**：查看课程掌握度、专题进度、复习时长、连续学习和模型探索情况。

## 数据备份

在“我的”页面点击“导出备份”，即可保存 JSON 备份文件。需要恢复时点击“导入备份”。清理浏览器数据前建议先导出备份。

## 目录说明

- `生物学考复习.html`：可直接双击打开的单文件成品。
- `离线程序/`：常规构建版本。
- `src/`：课程数据与程序源文件，便于以后补充内容。

教材知识依据用户提供的浙科版《分子与细胞》和《遗传与进化》整理；“浙江学考提示”为复习建议，不替代学校最新教学安排。
